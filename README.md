# Vue.js-Uikit3

> Vue.js Components with Uikit3

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Override Uikit Custom theme
- theme
    - uikit.app-theme.scss
